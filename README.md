# Deep Learning for Computer Vision

## About this repository
This repository documents my work in deep learning and computer vision using Python, with a focus on practical implementation and experimentation.

It includes:

- Implementation of neural networks using PyTorch (model design, training, and evaluation)
- Computer vision algorithms using OpenCV (image processing, feature detection, and video analysis)
- Small-scale projects exploring classification, segmentation, detection, and visual data processing
- Experiments with model performance, hyperparameters, and data preprocessing

## Topics covered
- PyTorch fundamentals (tensors, autograd, training loops)
- Neural network architectures for classification tasks
- Data preprocessing and augmentation
- Visualization of model performance and decision boundaries
- Classical computer vision techniques (OpenCV)

## Status
This repository is continuously updated with new projects and experiments.
